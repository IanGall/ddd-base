# gateway-parent 模块协作说明

## 模块定位
- 本模块是网关治理父工程，统一维护插件管理与构建参数。
- 负责为下游网关模块提供一致的编译、测试与打包基线。

## 变更边界
- 允许修改：pluginManagement、父级构建参数、统一构建约束。
- 禁止修改：具体业务逻辑、领域规则与运行时功能实现。

## 依赖约束
- 插件版本需集中治理，避免子模块分散覆盖。
- 保持构建参数可复用与可继承，减少重复配置。

## 提交前检查
- 校验插件版本与参数定义是否一致且可继承。
- 确认编译与测试插件配置不会破坏下游默认行为。
- 清理未使用插件配置与冗余构建属性。

# gateway-platform-bom 模块协作说明

## 模块定位
- 本模块负责网关平台依赖 BOM 聚合，统一平台技术栈版本。
- 聚合 Spring Boot、ddd-base-bom 与标准框架 BOM 依赖入口。

## 变更边界
- 允许修改：平台 BOM 导入项、依赖版本对齐策略、版本属性。
- 禁止修改：业务代码实现与非依赖治理职责内容。

## 依赖约束
- 新增或升级依赖需优先评估兼容性与传递影响。
- 保持导入顺序和职责清晰，避免重复或冲突管理。

## 提交前检查
- 核对 BOM 导入项完整性与版本一致性。
- 检查关键依赖升级是否可能引发二义性冲突。
- 删除废弃导入与无效版本配置项。
